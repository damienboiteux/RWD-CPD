# WorkHub — pack pédagogique v3

Ce pack contient :
- 5 pages principales : accueil, espaces, tarifs, événements, contact
- 1 page `auth.html` pour connexion / inscription sur la même page
- 1 page `demo-burger-css.html` pour la version burger CSS pur
- 1 version principale du burger en JavaScript (`js/menu.js`)
- 1 slider de témoignages (`js/slider.js`)
- 1 système d’onglets pour la page d’authentification (`js/auth-tabs.js`)
- 1 hero plein écran avec texte superposé
- des icônes Google Material Symbols pour Connexion / Inscription et les réseaux sociaux
- des tailles de police fluides via `clamp()`
- des images responsives avec `srcset`
- un footer à 3 colonnes : coordonnées, réseaux sociaux, newsletter

## Utilisation
Ouvrir `index.html` dans un navigateur.

## À faire compléter par les étudiants
- remplacer les SVG de démonstration par leurs propres visuels
- améliorer le slider ou le remplacer par une librairie externe
- enrichir la page connexion / inscription
- styliser davantage les états de formulaires
- faire évoluer la version CSS pur du burger puis la comparer avec la version JS
